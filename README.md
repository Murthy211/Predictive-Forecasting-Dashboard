
# Predictive Forecasting of Care Load and Placement Demand

## 📌 Project Overview

The **Predictive Forecasting of Care Load and Placement Demand** project is a machine learning-based web application that predicts future care load and placement demand using historical data. The application helps healthcare organizations and care management teams analyze trends, forecast demand, and support better resource planning.

The project is built using **Python**, **Streamlit**, and popular machine learning libraries such as **Scikit-learn** and **XGBoost**.

---

## 🚀 Features

* Interactive web interface using Streamlit
* Predictive analytics for care load forecasting
* Placement demand prediction
* Data visualization with charts and graphs
* Machine Learning model integration
* User-friendly dashboard

## ⚙️ Installation

### Clone the Repository

```bash
git clone https://github.com/your-username/predictive-forecasting-of-care-load-and-placement-demand.git
```

### Navigate to the Project Folder

```bash
cd predictive-forecasting-of-care-load-and-placement-demand
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

---

## ▶️ Run the Application

```bash
python -m streamlit run src/app1.py
```

After running the command, open your browser and visit:

```
http://localhost:8501
```
